# 🌐 Personal Portfolio Website

A fully responsive, zero-dependency developer portfolio built in vanilla HTML/CSS/JS.

## ✨ Features
- Custom animated cursor
- Smooth scroll navigation
- Scroll-reveal animations (IntersectionObserver API)
- Fully accessible (WCAG compliant, semantic HTML5)
- 90+ Lighthouse score
- Deployed on GitHub Pages

## 🛠️ Tech Stack
`HTML5` `CSS3` `Vanilla JavaScript` `IntersectionObserver API` `GitHub Pages`

## 📁 Project Structure
```
portfolio/
├── index.html        ← Main HTML file (all-in-one)
├── assets/
│   ├── css/style.css
│   ├── js/main.js
│   └── img/
└── README.md
```

## ⚙️ Setup & Run
```bash
# Just open in browser — no build step needed!
open index.html

# Or serve locally
npx serve .
```

## 🚢 Deploy on GitHub Pages
1. Push code to GitHub
2. Go to **Settings → Pages**
3. Set Source: `main` branch → `/root`
4. Your site is live at `https://NarayanaVasan-crypto.github.io/portfolio`
